# 🧮 Calculadora Simples

Este é meu primeiro projeto desenvolvido no curso de Análise e Desenvolvimento de Sistemas.

Decidi criar uma calculadora simples para praticar lógica de programação e também aprender mais sobre HTML, CSS e JavaScript.

---

## 🚀 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

---

## 📌 Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Botão para limpar (C)
- Botão para apagar último número (⌫)
- Tratamento básico de erro
- Interface responsiva e centralizada

---

## 🎯 Objetivo do projeto

O objetivo foi:

- Praticar estrutura de páginas HTML
- Estilização com CSS
- Manipulação do DOM com JavaScript
- Trabalhar com funções
- Entender melhor a lógica por trás de uma calculadora

---

## 🖥️ Como executar o projeto

1. Baixe os arquivos
2. Abra o arquivo `index.html` no navegador

Ou acesse pelo GitHub Pages (caso esteja publicado).

---

## 📚 O que eu aprendi

Durante esse projeto eu aprendi:

- Como organizar um projeto simples
- Como conectar HTML, CSS e JavaScript
- Como criar funções para manipular elementos na tela
